# Portfolio — Ximena Alicia Talavera Miranda

Portafolio profesional de Ximena Alicia Talavera Miranda.

**Live:** [cofoundy.github.io/portfolio-ximena-talavera](https://cofoundy.github.io/portfolio-ximena-talavera)

## Tech Stack

- [Astro](https://astro.build/) — Framework
- [Tailwind CSS](https://tailwindcss.com/) — Estilos
- [TypeScript](https://www.typescriptlang.org/) — Tipado

## Editar contenido

Todos los datos del portafolio están en `src/config.ts`. Edita ese archivo para actualizar tu información.

```
src/
├── config.ts          # ← Tus datos van aquí
├── components/        # Componentes visuales
├── layouts/           # Layout principal
└── pages/             # Páginas
public/
├── profile.jpg        # Foto de perfil
└── favicon.svg        # Ícono del sitio
```

## Desarrollo local

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # Generar sitio estático
```
